# Delta programme scolaire CP→CM2

Sources officielles principales :
- `education.gouv.fr` : programmes et horaires de l'école élémentaire
- `eduscol.education.fr` : accompagnements cycle 2 et cycle 3

## Méthode

Le delta compare le contenu actuel de l'application aux attentes du programme du `CP` au `CM2`.

Statuts utilisés :
- `Couvert`
- `Partiel`
- `Manquant`
- `À corriger`
- `À enrichir`

Lecture spécifique pour les leçons :
- `Leçon conforme`
- `Leçon trop méthodologique`
- `Leçon à réécrire`
- `Leçon absente`

## Ce qui est déjà solide

- structure CP→CM2 cohérente
- bibliothèque de leçons en place
- corpus d'exercices large et exploitable
- forte couverture en français et mathématiques
- socle documentaire désormais présent sur tous les niveaux

## Delta majeur restant

### 1. Leçons

Le plus gros delta n'est plus technique, mais pédagogique :
- certaines leçons restent trop procédurales
- certaines leçons documentaires restent trop méthodologiques
- plusieurs zones doivent encore ressembler davantage à une mini-fiche de manuel

### 2. Mathématiques

Priorités transverses :
- géométrie
- problèmes
- grandeurs et mesures
- représentation et organisation de données

### 3. Français

Priorités transverses :
- lecture-compréhension
- vocabulaire structuré
- syntaxe et phrase
- production écrite

### 4. Documentaires

Priorités transverses :
- histoire plus narrative
- géographie plus ancrée dans les territoires et les espaces vécus
- sciences avec davantage de vocabulaire disciplinaire
- EMC avec institutions, laïcité, loi, justice, égalité

## Priorisation

### P1

- réécrire les leçons non conformes pédagogiquement
- corriger les reliquats de langue et d'encodage dans les niveaux
- compléter les notions du programme absentes dans les bibliothèques de leçons

### P2

- épaissir les catégories encore trop légères
- densifier géométrie, problèmes, vocabulaire, lecture-compréhension
- enrichir les sciences et l'EMC disciplinaires

### P3

- harmoniser l'éditorial et l'UX des leçons
- renforcer la bibliothèque globale des leçons
- préparer une progression mieux suivie
