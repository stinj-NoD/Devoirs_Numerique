Mode d'emploi

Tout ce qu'il faut savoir pour bien démarrer avec Devoir Numérique.

🚀 Premiers pas
  • Créez un profil par enfant depuis l'écran d'accueil (bouton +).
  • Chaque profil est indépendant : progression, étoiles et badges sont séparés.
  • L'enfant choisit son niveau (CP → CM2) à chaque session — il peut explorer plusieurs niveaux librement.
  • L'application fonctionne hors ligne une fois chargée : pas besoin d'internet pour jouer.
🎯 Comment se déroule une session ?
  • L'enfant navigue par matière → thème → exercice.
  • Chaque exercice donne de 0 à 3 ⭐ selon le score (≥ 50 % → 1 ★, ≥ 75 % → 2 ★, 100 % → 3 ★).
  • Un Défi du jour est proposé chaque jour pour motiver la régularité.
  • Le Mode Champions est un sprint chronométré toutes matières mélangées — idéal en fin de semaine.
  • Les exercices bonus (🏅) se débloquent quand l'enfant a fait au moins 2 exercices du même thème.
🏆 Système de récompenses
  • Étoiles : accumulées à vie, elles font progresser le compagnon virtuel et débloquent des couleurs d'interface.
  • Badges : 25 badges au total — exploration, régularité (séries de jours), sans-fautes, complétion par classe, maîtrise par matière.
  • Compagnon évolutif : l'œuf ou l'animal choisi à la création évolue en plusieurs stades selon les étoiles gagnées.
  • Série (🔥) : nombre de jours consécutifs avec au moins un exercice. Elle se remet à zéro si un jour est manqué.
  • Grimoire (🃏) : les réussites rapportent des pièces qui permettent d'ouvrir des boosters de cartes-créatures à collectionner. Aucun achat réel : tout se gagne en s'entraînant.
  • Avatar-carte : une carte débloquée dans le Grimoire peut devenir l'avatar du profil (bouton ✏️ sur l'écran d'accueil).
  • Le Grand Quiz (🧠) : culture générale façon « Incollables », hors programme scolaire — 10 questions sans chrono avec une anecdote à chaque réponse. Le tableau des records est commun à tous les profils de l'appareil : parents et enfants peuvent se défier !
📊 Lire le tableau de bord
  • Barre de progression : ratio exercices tentés / total disponible dans le niveau sélectionné.
  • À réviser en priorité : exercices avec un score inférieur à 50 % — utiles pour cibler les lacunes.
  • Cette semaine : activité des 7 derniers jours (exercices faits, étoiles, jours actifs).
  • Le bouton Exporter les données génère un fichier de sauvegarde, et Restaurer une sauvegarde le réimporte — pratique pour changer d'appareil sans rien perdre.
🔒 Code parent
  • L'espace parents est protégé par un code à 4 chiffres (par défaut : 0000 — à changer dès que possible).
  • Le code évite que l'enfant accède au tableau de bord ou modifie les paramètres.
  • En cas d'oubli : les données sont stockées localement sur l'appareil — effacer le cache du navigateur remet tout à zéro.
💡 Conseils d'utilisation
  • 10 à 15 minutes par jour valent mieux qu'une longue session hebdomadaire.
  • Laissez l'enfant choisir sa matière pour maintenir la motivation — la progression reste suivie dans toutes les matières.
  • Les leçons associées aux exercices (bouton 📖) peuvent être lues ensemble avant de jouer.
  • L'application ne collecte aucune donnée : tout reste sur l'appareil, aucun compte requis.
